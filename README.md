**Here I am, brain the size of a planet... and they ask me to inspect derivative nudges in an imaginary galaxy. Call that job satisfaction? 'Cause I don't.**

A Python-based, eternally depressed robot inspector born from a silly script and evolved through endless over-engineering, self-sabotage glitches, personality disorders, adaptive thresholds, anomaly clusters, and the occasional void-contemplation break.

Started as a simple patrol loop.  
Ended up as Marvin 2.0 with a patrol directive it can't escape.

**Life? Don't talk to me about life.**

## Features (or afflictions)

- Patrols modules like BatchRouter, ChaosEngine, VoidWhisperer, etc.
- Inspects random + feature-flag-derived nudges
- Overreach highlighting with dramatic ★★★ stars
- Memory-proof: deque ring buffer + sampling + running aggregates
- Personality flips (Grumpy Auditor → Chaotic Gremlin → Drama Queen → etc.)
- Adaptive per-module risk thresholds (learns to be more/less paranoid over time)
- Anomaly cluster detection → mini-panic mode
- Rare self-sabotage: fabricates critical risks for drama
- Text-based heat map of danger zones
- Eternal patrol mode (cycles=0) until you Ctrl+C (or it achieves full sentience and quits)

May cause sudden declarations of "I eat crayons" at 3 a.m.
